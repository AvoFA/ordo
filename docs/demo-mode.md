# Ordo Demo Mode

Demo Mode — это специальный отладочный режим для локальной разработки, который позволяет запускать и просматривать интерфейс Ordo без подключенной и запущенной базы данных PostgreSQL.

## Как включить

В файле `.env` добавьте переменную:

```env
ORDO_DEMO_MODE=true
```

## Как это работает

1. **Обход авторизации (только в Development)**:
   При запросе страниц рабочего пространства (`/`, `/learning-paths` и т.д.) вместо перенаправления на `/sign-in` автоматически предоставляется демонстрационный сессионный пользователь `demo@ordo.app`.
   
2. **Имитация данных (Mock-fallbacks)**:
   Все обращения к репозиториям данных оборачиваются в безопасную обертку. Если база данных недоступна, приложение автоматически переключается на использование локальных статических слепков данных (эквивалентных тем, что генерируются сидом).

## Безопасность

> [!WARNING]
> Переменная `ORDO_DEMO_MODE=true` работает **исключительно** при значении переменной окружения `NODE_ENV === 'development'`.
> В режиме production (`NODE_ENV === 'production'`) этот режим технически отключен, и обход авторизации заблокирован.
